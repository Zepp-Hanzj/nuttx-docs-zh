==================================
``sensorscope`` 传感器监视工具
==================================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

``sensorscope`` 是一个用于监视和显示传感器数据的工具。

配置选项
--------

- ``CONFIG_SYSTEM_SENSORSCOPE`` – 启用 sensorscope 应用程序。
- ``CONFIG_SENSORS`` – 启用传感器子系统支持。
