===================================
``nxffs`` NXFFS file system example
===================================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

这是 NuttX NXFFS FLASH 文件系统的测试。这是一个 NXFFS 压力测试，对文件系统进行了非常严格的测试。它应该只在模拟环境中使用！将此 NXFFS 测试放在真实硬件上很可能会损坏您的 FLASH。您已被警告。
