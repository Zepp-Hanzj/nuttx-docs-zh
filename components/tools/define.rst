=============================
``define.sh``, ``define.bat``
=============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

Different 编译rs have different conventions for specifying pre-
processor definitions on the 编译r command line.  This bash
script allows the 构建 system to 创建 command line definitions
without concern for the particular 编译r in use.

The define.bat script is a counterpart for use in the native 窗口s
构建.
