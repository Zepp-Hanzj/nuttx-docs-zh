=================================
``telnetc`` Telnet 客户端库
=================================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

这是 libtelnet 到 NuttX 的移植。这是一个公共领域的 Telnet 客户端库，可从 https://github.com/seanmiddleditch/libtelnet 获取，已修改以用于 NuttX。
