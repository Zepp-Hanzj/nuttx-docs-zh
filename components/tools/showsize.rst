===============
``showsize.sh``
===============

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

Show the top 10 biggest 内存 hogs in code and 数据 spaces. This must be
执行d from the top-level NuttX 目录 like:

.. code:: console

   $ tools/showsize.sh
   TOP 10 BIG DATA
   ...
   TOP 10 BIG CODE
   ...
