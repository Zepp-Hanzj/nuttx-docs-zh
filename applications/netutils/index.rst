.. rubric:: uIP 应用程序

本目录包含 ``uIP-1.0`` apps 目录下的大部分网络应用程序。正如 uIP ``apps/README.md`` 所述，这些应用程序"并非全部经过充分测试"。这些基于 uIP 的应用包括：``dhcpc``、``smtp``、
``webclient`` 和 ``webserver``

您可以通过以下链接在 uIP 论坛中找到有关这些应用程序的更多信息：
http://www.sics.se/~adam/uip/index.php/Main_Page。其中一些应用程序（例如 uIP Web 服务器）由于主要来自 NuttX 用户的贡献，已经增加了一些额外功能。
