=========================================================================
``Libraries.mk``, ``FlatLibs.mk``, ``ProtectedLibs.mk``, ``KernelLib.mk``
=========================================================================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

Libraries.mk has the 构建 rules for all NuttX libraries.

FlatLibs.mk, ProtectedLibs.mk, and KernelLib.mk:  These control the
selection of libraries to be built, depending on the selected 构建 mode.
