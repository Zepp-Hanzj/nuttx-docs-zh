=======
Testing
=======

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

``apps/testing`` 目录用于构建 NuttX 特定的测试以及包含外部测试框架。

``apps/examples`` 和 ``apps/testing`` 之间存在一定的重叠，因为 ``apps/examples`` 中也有一些测试。不过，那些测试还可以用来展示 NuttX 功能的使用方法。而 ``apps/testing`` 中的大多数测试则是纯粹的测试，作为使用示例的价值不大。
