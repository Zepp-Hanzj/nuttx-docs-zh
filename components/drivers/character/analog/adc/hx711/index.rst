========================
Avia Semiconductor HX711
========================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

由 Michał Łyszczek 贡献的驱动程序。

HX711 是一款专为体重秤设计的 24 位 ADC（模数转换器）。这款芯片可能非常慢。使用内部振荡器且 RATE 引脚拉低时，它每秒仅输出 10 个采样值。为了不占用 CPU，驱动程序使用中断来检测芯片何时准备好。这将使 read(2) 变为阻塞操作，但系统可以在芯片准备好之前执行其他任务。因此，驱动程序没有完全遵循 ADC API，而是使用标准字符设备接口（只读）。

可以使用 ``cat`` 命令从 shell 中轻松读取应变传感器的值

.. code-block::

   cat /dev/hxx711_0

不过使用示例程序 ``hx711`` 转储值可能更好，因为 ``cat`` 会一直读取下去，如果 ctrl+c 不起作用，它将永远占用 shell。

-------
读取
-------

读取通过调用标准 POSIX read(2) 函数完成。每次调用 read(2) 只能返回一个值。但可以启用平均功能，使驱动程序读取 N 个采样值，取平均值，然后返回单个平均值。

此函数接受两种类型的缓冲区。

如果缓冲区大小为 ``sizeof(int32_t)``，则 int32 值将存储在缓冲区中。如果缓冲区大小大于 ``sizeof(int32_t)``，函数将把值的字符串表示形式存储在传入的缓冲区中。

读取并打印值的简单代码如下

.. code-block:: C

   int fd;
   fd = open("/dev/hx711_0", O_RDONLY);

   for (; ; )
     {
       int32_t value;
       value = read(fd, &value, sizeof(value));
       printf("Read: %"PRIi32"\n", value);
     }

-----
ioctl
-----

由于此芯片（和驱动程序）专为体重秤设计，内核驱动程序可以提供一些处理功能，使用户空间代码更简单。这些功能通过 ioctl(2) 命令实现。实际上，这些功能都不是必需的，但如果您只是打开驱动程序并读取它，您将获得 hx711 芯片的原始值，需要自行处理。如果您的需求更标准，最好使用内核处理功能。

HX711_SET_AVERAGE
-----------------

.. code-block:: C

   unsigned average = 5;
   ioctl(fd, HX711_SET_AVERAGE, average);

驱动程序将从 hx711 读取指定数量的采样值，并返回它们的平均值。为避免整数溢出导致数据损坏，最大平均值设置为 225。如果需要对更多值取平均值，您需要编写自己的代码。

HX711_SET_CHANNEL
-----------------

.. code-block:: C

   char channel = 'a';
   ioctl(fd, HX711_SET_CHANNEL, channel);

HX711 有 2 个通道，A 和 B，可以根据需要切换。驱动程序自动执行虚拟读取，使下一次 read(2) 调用返回新通道的值。当切换到通道 'B' 时，驱动程序自动将增益更改为 32（唯一可能的值）。切换回 'A' 将把增益设置为 128。

HX711_SET_GAIN
--------------

.. code-block:: C

   unsigned char gain = 128;
   ioctl(fd, HX711_SET_GAIN, gain);

设置增益。通道 'A' 支持增益 "128" 和 "64"。通道 'B' 只有一个增益选项 - 32。

HX711_SET_VAL_PER_UNIT
----------------------

.. code-block:: C

   int val_per_unit = 813;
   ioctl(fd, HX711_SET_VAL_PER_UNIT, val_per_unit);

驱动程序可以执行计算，使您可以读取物理值，如克、盎司或磅，或者您自己的自定义单位。您只需指定应变传感器的哪个值对应一个单位。

假设您有一个最大值为 1,000,000 的应变传感器。值 100,000 表示 1kg，传感器完全线性。如果您想以 kg 为单位获取读数，您可以将 ``val_per_unit`` 设置为 100,000。如果您想要以克为单位输出，它将是值 100。要获得十分之一克的精度，您可以将其设置为 10。驱动程序不关心单位，您只需选择一个并坚持使用。

请注意，驱动程序只能返回整数，因此如果您将其设置为返回 kg 单位，您将只能获得 1、2、3kg...，而无法感知 0.5kg 或 1.5kg。为此，您需要将值设置为 10,000，驱动程序将返回 15（对应 1.5kg）或 0.5（对应 0.5kg）的值。

HX711_TARE
----------

.. code-block:: C

  float precision = 0.1;
  ioctl(fd, HX711_TARE, &precision);

每个秤都需要去皮功能。驱动程序轮询 hx711 一段时间，如果检测到秤处于稳定状态，ioctl(2) 将成功返回，下一次 read(2) 调用将在返回读数时考虑新的去皮值。当几次连续读数（最小-最大值）在指定精度范围内时，秤被假定为稳定。

如果之前设置了 ``HX711_SET_VAL_PER_UNIT``，您可以使用您的单位传递值。如果您将驱动程序配置为以克为单位工作，您可以将此值设置为 0.1（克）或 5（克）。

如果驱动程序在一定时间内无法获得稳定读数，它将返回 ETIME 错误。

重要提示，请确保在去皮之前设置了正确的符号，否则您将加倍去皮值而不是将其归零！

HX711_SIGN
----------

.. code-block:: C

   int sign = -1;
   ioctl(fd, HX711_SIGN, &sign);

如果当秤上的质量增加时驱动程序的值降低，您可以交换符号。当应变传感器安装反了时，这可能是必要的。

---------------------
hx711 示例程序
---------------------

在应用配置 ---> 示例中还有一个名为 ``HX711 driver example`` 的配套程序。其主要目的是展示如何使用驱动程序，但它也是从 shell 快速调试芯片的非常好的工具，因为它可以转储读数并设置所有选项。

.. code-block::

   -h       打印此帮助信息
   -d<path> hx711 设备路径，默认：/dev/hx711_0
   -t<prec> 以指定精度对秤去皮，可能需要几秒钟才能完成。
            如果设置了每单位值，精度以单位为单位，否则为原始值。
            如果使用单位，可以传递浮点数，如 0.1
   -v<val>  对应于一个单位的读数值。此值需要先校准才能知道
   -s       反转符号，如果当质量增加时值降低，请传递此选项
   -D       转储当前设备设置（如平均值、通道、增益等）
   -a<avg>  设置返回值前应平均的采样数，有效值 [1..225]
   -c<chan> 设置要读取的通道（'a' 或 'b' 有效）
   -g<gain> 设置 adc 增益，对于通道 'a'，64 和 128 有效，
            对于通道 'b'，只有 64 有效
   -r<num>  退出前读取此数量的采样值，采样值将以字符串形式打印在
            标准输出上，每行一个采样值

   设置的值是持久的，即一旦设置就存储在驱动程序中，并在此程序执行期间应用。

   如果您只指定 <-a|-c|-g|-v|-t> 而不指定 -r，程序将设置新参数并退出。
   您可以稍后仅使用 -r 选项再次调用程序，以使用先前设置的值读取采样值。
   您也可以在一次调用中传递所有选项

   要测试是否需要设置 CONFIG_ADC_HX711_ADD_DELAY 选项，请运行：
        hx711 -a225 -r128
   这将使 hx711 芯片加载足够长的时间以显示由于缺少添加的延迟而导致的任何可能错误。

   程序按顺序执行：设置选项、去皮、转储、运行，因此如果您指定所有选项，
   新设置将被应用，然后新设置将被打印，最后程序将对秤去皮并打印采样值

   示例：

   为第一个芯片设置 hx711 设置并退出：
        hx711 -a32 -ca -g64

   从不同芯片转储芯片设置
        hx711 -d/dev/hx711_2 -D

   使用先前设置的 hx711 设置读取 10 个采样值
        hx711 -r10

   更改通道并读取 32 个采样值（平均设置不会改变）：
        hx711 -cb -r32

   设置每单位值，以克为单位获取输出，然后以 10g 精度去皮
        hx711 -v 813 -t 10
