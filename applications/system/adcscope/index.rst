==============================
``adcscope`` ADC 作用域工具
==============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

``adcscope`` 是一个用于监视和显示 ADC（模数转换器）数据的工具。

配置选项
--------

- ``CONFIG_SYSTEM_ADCSCOPE`` – 启用 adcscope 应用程序。
- ``CONFIG_ADC`` – 启用 ADC 子系统支持。
