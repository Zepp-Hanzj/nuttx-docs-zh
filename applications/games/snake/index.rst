========================
``snake`` 贪吃蛇
========================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

贪吃蛇是一款经典游戏，在旧手机上特别受欢迎。游戏在一个矩阵（例如 6x6、8x8）上进行，
用不同颜色的方块（单元格）表示蛇和食物对象。

游戏的目标是吃到尽可能多的食物而不撞到自己。吃到食物后，蛇的尾巴增加一节，
直到蛇的大小等于游戏棋盘的大小。

游戏开始时蛇的长度为一，玩家可以穿过棋盘的边缘。当蛇到达边缘时，它会出现在对面。

基本测试
----------

要玩游戏，您需要一个 LED 矩阵（例如 ws2812）用于输出，四个 GPIO 引脚用于输入。

或者，您可以通过更改设置使用串行控制台输入。

然后您可以配置并编译游戏以在您的开发板上玩，
例如，对于 ESP32-Devkitc，已经有一个示例::


    $ ./tools/configure.sh esp32-devkitc:snake
    $ make -j flash ESPTOOL_PORT=/dev/ttyUSB0
    $ minicom
    nsh> snake
