========================
``brickmatch`` 砖块匹配
========================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

砖块匹配是一个具有挑战性的游戏，玩家通过将相同颜色的砖块匹配来获得分数。
一个有趣的事实是，NuttX 徽标的灵感来自于砖块匹配游戏中可以使用的灯光矩阵。

在游戏矩阵（例如 6x6 或 8x8）中，每个块可以是不同颜色的。如果相邻块具有相同的颜色，
该块将闪烁并从棋盘上移除，为移动留出更多空间。

游戏开始时只有边框单元格被填充，您可以将墙壁、地板和天花板向棋盘中心方向移动，
以使相同颜色的单元格匹配。

基本测试
----------

玩砖块匹配最好的方式是使用 APA102 RGB 16x16 矩阵和手势传感器 APDS9960。
已经有一些开发板示例完成了此集成，您只需将 APA102 矩阵连接到正确的 SPI 引脚
（查看您的开发板配置）并将 APDS9960 连接到 I2C 端口（同时连接其 INT 引脚）。

如果您没有 APA102 矩阵，也可以使用 LCD 显示屏或 LED 矩阵（WS2812）
和数字操纵杆（DJOYSTICK）、GPIO 引脚或控制台输入来玩。

然后您可以配置并编译游戏以在您的开发板上玩，
例如，对于 ESP32-Devkitc，已经有一个使用 APA102 的示例::


    $ ./tools/configure.sh esp32-devkitc:brickmatch
    $ make -j flash ESPTOOL_PORT=/dev/ttyUSB0
    $ minicom
    nsh> brick

就是这样！玩得开心！
