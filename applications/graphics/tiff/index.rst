==============================
``tiff`` TIFF 创建库
==============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

本目录包含一个可用于创建 TIFF 图像文件的库。创建此文件的目的是支持从 LCD 进行屏幕截图。不过，该逻辑是通用的，可以用于大多数用途。

唯一的使用文档在文件 ``apps/include/tiff.h`` 中（相当详细的）注释中。

单元测试
---------

参见 ``apps/examples/tiff``。
