=========
资源
=========

.. note:: 此内容应修订

以下是您可能会发现有用的 Apache NuttX 资源列表：

 * Apache NuttX

   * `Apache NuttX 网站 <https://nuttx.apache.org>`_
   * `Apache NuttX 在线文档 <https://cwiki.apache.org/confluence/display/NUTTX/Nuttx>`_
   * `Apache NuttX 邮件列表 <https://nuttx.apache.org/community/>`_ – 一个非常活跃的邮件列表，是获取应用程序帮助或有关 NuttX 的任何问题的地方。
   * `Apache NuttX YouTube 频道 <https://www.youtube.com/channel/UC0QciIlcUnjJkL5yJJBmluw/videos>`_ – Alan Carvalho de Assis 的 NuttX YouTube 频道。这是大量实用信息的来源。
   * `Apache NuttX 编码标准 <https://cwiki.apache.org/confluence/display/NUTTX/Coding+Standard>`_ — 提交新文件或修改现有文件时代码应遵循的规范。
   * `Apache NuttX 代码贡献指南 <https://cwiki.apache.org/confluence/display/NUTTX/Code+Contribution+Workflow>`_ — 提交代码的完整工作流程及所有细节。

 * Git

   * `Git 速查表（GitHub） <https://github.github.com/training-kit/downloads/github-git-cheat-sheet.pdf>`_
   * `Git 手册（在线） <https://git-scm.com/book/en/v2>`_
