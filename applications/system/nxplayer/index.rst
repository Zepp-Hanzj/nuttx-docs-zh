==============================
``nxplayer`` nxplayer 播放器
==============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

``nxplayer`` 是一个命令行媒体播放器应用程序，可用于播放音频文件。

配置选项
--------

- ``CONFIG_SYSTEM_NXPLAYER`` – 启用 nxplayer 应用程序。
- ``CONFIG_AUDIO`` – 启用音频子系统支持。
