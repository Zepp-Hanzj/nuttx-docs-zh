===============================
Data Structures Category (NUTS)
===============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

此测试类别测试内核中用于多个其他 API 的 NuttX 数据结构的行为。可以使用 ``CONFIG_TESTING_NUTS_DSTRUCTS`` 启用该类别。

以下数据结构有测试套件：

* ``CONFIG_TESTING_NUTS_DSTRUCTS``: ``struct circbuf_s`` 的测试
* ``CONFIG_TESTING_NUTS_DSTRUCTS``: ``struct list_node`` 的测试
