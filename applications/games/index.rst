=====
游戏
=====

NuttX 不仅仅是一个实时操作系统，它还是一个充满乐趣的丰富生态系统。
对很多人来说，乐趣与游戏息息相关！那么，让我们开始玩吧！

在这里，您将找到 NuttX 当前支持的游戏列表。

一个有趣的事实是，NuttX 最初有一个名为 Traveler 的游戏。
这是一个由 Greg Nutt 从零开始开发的光线投射游戏，在很多方面类似于 Doom。
该游戏已不再是 Apache NuttX 的一部分。
