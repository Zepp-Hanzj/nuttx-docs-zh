=======================
``cjson`` cJSON 库
=======================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

cJSON 是一个超轻量级、可移植、单文件、尽可能简单的符合 ANSI-C 标准的 JSON 解析器，采用 MIT 许可证。可嵌入的轻量级 XML-RPC 服务器讨论请参阅
http://www.drdobbs.com/web-development/an-embeddable-lightweight-xml-rpc-server/184405364。

此代码取自 http://sourceforge.net/projects/cjson/，并由 Darcy Gong 为 NuttX 进行了适配。
