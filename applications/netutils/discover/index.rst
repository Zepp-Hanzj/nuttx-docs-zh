======================================
``discover`` 网络发现工具
======================================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

此守护进程可用于发现本地网络中的设备，特别是配置了 DHCP 的设备。它监听 UDP 广播，广播中还可以包含设备类别，以便发现设备组。也可以使用某种广播发现方式来寻址所有类别。

有关客户端示例，请参阅 `nuttx/tools/discover.py`。
