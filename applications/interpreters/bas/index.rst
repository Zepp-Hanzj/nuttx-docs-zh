=============================
``bas`` Bas BASIC 解释器
=============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

简介
------------

Bas 是经典 BASIC 编程语言方言的解释器。它与 1980 年代典型的 BASIC 解释器相当兼容，
不像其他一些 UNIX BASIC 解释器实现了不同的语法，破坏了与现有程序的兼容性。
Bas 提供了许多用于结构化编程的 ANSI BASIC 语句，如过程、局部变量和各种循环类型。
此外还有矩阵运算、自动 LIST 缩进以及许多特定经典方言中的语句和函数。行号不是必需的。

解释器在运行程序之前会对源代码进行分词并解析变量和跳转目标的引用。
此编译过程提高了效率，并捕获了语法错误、类型错误和从未初始化的变量引用。
Bas 使用 ANSI C 为 UNIX 系统编写。

Bas 2.4 发布说明
---------------------

与 ``2.3`` 版本相比的变更

- 修复了选项基数为 1 的整数数组上的矩阵求逆。
- 修复了 ``!`` 的 ``PRINT USING`` 行为。
- ``PRINT`` 分隔符应前进到下一个区域，即使当前位置在区域的开头。
- 添加了 ``ip()``、``frac()``、``fp()``、``log10()``、``log2()``、``min()`` 和 ``max()``。
- 修复了 ``NEXT`` 区分大小写检查变量的问题。
- 使用 ``terminfo`` 功能 cr 以利用其填充。
- 修复了 ``LET`` 段错误。
- ``PRINT`` 现在使用打印项。
- ``-r`` 用于受限操作。
- ``MAT INPUT`` 不会丢弃多余的参数，而是将它们用于下一行。
- 许可证更改为 MIT。
