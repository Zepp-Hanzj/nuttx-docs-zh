==========================
``ping`` ICMP ping 支持
==========================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

这是一个使用原始套接字的 ping 和 ping6 的未完成实现。它尚未集成到配置或构建系统中。

当前 NSH 中的 ``ping`` / ``ping6`` 逻辑为了实现 ``ping`` / ``ping6`` 而对操作系统进行了非法调用。一种正确的实现方式是使用原始套接字将 ``ping`` / ``ping6`` 实现为用户应用程序。这是此类实现的初步尝试。
