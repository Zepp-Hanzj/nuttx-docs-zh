============================
``wapi`` WAPI (Wireless API)
============================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

可以使用 ``wapi`` 连接到接入点、配置接入点、设置 Wi-Fi 相关配置并获取相关信息。

目前，``wapi`` 是唯一提供的从用户空间处理 Wi-Fi 网络的接口。
