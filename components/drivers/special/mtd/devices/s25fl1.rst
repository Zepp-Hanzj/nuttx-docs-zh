================
S25FL1 NOR Flash
================

.. note:: 本文档翻译自 NuttX 官方文档，如需查阅最新版本请访问 https://nuttx.apache.org/docs/latest/

NuttX 为 S25FL1 系列的 NOR Flash 提供支持。支持最高 64 Mbit
（8 MB）的容量。

基于 QSPI 的 Flash 通过选项 ``CONFIG_MTD_S25FL1`` 启用。可以
通过 ``CONFIG_S25FL1_QSPIMODE`` 选项选择 QSPI 模式，并通过
``CONFIG_S25FL1_QSPI_FREQUENCY`` 选项选择通信频率。

如果启用了 ``CONFIG_S25FL1_SECTOR512`` 选项，该 Flash 允许模拟
512 字节的大擦除块。可以通过 ``CONFIG_S25FL1_SCRAMBLE`` 启用数据
加扰。这需要驱动程序支持数据加扰/解扰。加扰密钥由
``CONFIG_S25FL1_SCRAMBLE_KEY`` 选项设置。

Flash 的大小在设备初始化期间自动设置。

闪存在使用前必须初始化。这通常在板卡启动阶段从板级支持包层完成。
此操作由以下函数执行。

.. code-block:: C

   #include <nuttx/mtd/mtd.h>

   FAR struct mtd_dev_s *s25fl1_initialize(FAR struct qspi_dev_s *qspi,
                                           bool unprotect)
