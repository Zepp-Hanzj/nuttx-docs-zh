====================================
``foc`` FOC 电机控制器示例
====================================

本示例的主要目的是提供一个通用模板，用于基于内核侧 FOC 设备和应用侧 FOC 库实现电机控制器。

目前，本示例实现了一个简单的开环速度控制器。

硬件设置
--------

本示例尚未实现任何保护供电设备的机制。这意味着没有过温保护、过流保护和过压保护。

请确保正确供电并自行提供电流限制，以免损坏硬件。

配置
----

FOC PI 电流控制器参数可由以下公式得出：

  Kp = ccb * Ls;
  pp = Rs / Ls;
  Ki = pp * Kp * T;

其中：
  - Kp  - PI 比例系数
  - Ki  - PI 积分系数
  - Rs  - 平均相串联电阻
  - Ls  - 平均相串联电感
  - pp  - 极点
  - ccb - 电流控制带宽
  - T   - 采样周期
